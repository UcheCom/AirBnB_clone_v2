# Project: 0x04. AirBnB clone - Web framework
# The goal of the project is to deploy on your server a simple copy of the AirBnB website. Templates will be used. In this project, the first web server in Python will be created and then the static HTML file willbe made dynamic by using objects stored in a file or database.
